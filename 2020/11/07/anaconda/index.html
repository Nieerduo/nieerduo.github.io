windows安装anaconda

1.到官网下载安装包。。

一.管理conda

windos用户打开“anaconda prompt”

验证conda已被安装

conda --version

更新conda至最新版本

conda update conda

二.管理环境

windows用户打开“Anaconda prompt”

创建新环境

conda create --name <env_name> <package_names>

切换环境

activate <env_name>

退出环境至root

deactivate

显示已创建的环境

conda info --envs

复制环境

conda create --name <new_env_name> --clone <copied_env_name>

删除环境

conda remove --name <env_name> --all

三.管理包

1.查找可供安装的包版本

精确查找
conda search --full-name <package_full_name>
模糊查找
conda search <text>

2.获取当前环境中已安装的包信息

conda list

3.安装包

在指定环境中安装包
conda install --name <env_name> <package_name>
在当前环境中安装包
conda install <package_name>
使用pip安装包
当使用conda install无法安装时，可以使用pip进行安装
pip install <package_name>
从anaconda.org安装包（未写明）

4.卸载包

卸载指定环境中的包
conda remove --name <env_name> <package_name>
卸载当前环境中的包

5.更新包

更新所有包
conda update --all
更新指定包
`conda update

清除清华源

conda config --remove-key channels
